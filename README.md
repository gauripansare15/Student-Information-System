# Student Information System (Java)

## Project Overview
The Student Information System is a console-based Java application used to manage student records.
It allows users to add, view, search, update, and delete student details using a simple menu-driven interface.

This project is developed to practice Core Java concepts and Object-Oriented Programming.

## Technologies Used
- Java (Core Java)
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner (for user input)

## Features
- Add new student information
- View all student records
- Search student by Student ID or Name
- Update student details
- Delete student records
- Input validation and basic error handling
- Menu-driven console application

## How to Run the Project
1. Open Command Prompt
2. Navigate to the src folder:
   cd Student-Information-System/src
3. Compile the Java files:
   javac *.java
4. Run the application:
   java StudentInformationSystem

## Sample Menu
1. Add Student
2. View Students
3. Search Student
4. Update Student
5. Delete Student
6. Exit
